# Events & Talks - Master File

**⚠️ IMPORTANT: This is your single source of truth for events, talks, and presentations.**
Edit this file to add/update events. Run the sync script to update the website.

---

## How to Use This File

1. **Add a new event** by creating a new entry below
2. **Format**: Follow the template structure shown
3. **Run sync**: After editing, run `python3 sync_events.py` (or manually copy to `content/event/`)
4. **Dates**: Use ISO format (YYYY-MM-DD or YYYY-MM-DDTHH:MM:SSZ)

---

## Event Entry Template

```markdown
### Event Name
- **Title:** Your Talk/Event Title
- **Event Type:** Research Presentation / Invited Talk / Conference / Workshop
- **Event URL:** https://example.com (optional)
- **Date:** YYYY-MM-DD or YYYY-MM-DDTHH:MM:SSZ
- **All Day:** true/false
- **Location:** Venue Name
- **Address:**
  - Street: Street Address
  - City: City
  - Region: State/Region
  - Postcode: ZIP Code
  - Country: Country Name
- **Summary:** One-line description
- **Abstract:** Detailed description of the talk/event
- **Tags:** tag1, tag2, tag3 (comma-separated)
- **Slides URL:** https://... (optional)
- **Video URL:** https://... (optional)
```
