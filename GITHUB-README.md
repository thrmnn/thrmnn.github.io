## Built

- **Roboat Perception Stack** — End-to-end vision pipeline for autonomous boats navigating Amsterdam's canals. Sub-30ms inference on Jetson Orin. First engineering hire at MIT spinoff.
- **Urban Tree Census Pipeline** — Aerial LiDAR + multispectral fusion for city-scale canopy density estimation. Automated LAI prediction via ML.
- **Favela Digital Twins** — Led 10-person team building 3D reconstructions from LiDAR + aerial imagery for public health CFD simulations. MIT Senseable City Lab, Rio de Janeiro.
- **ROS Autonomy Stack** — Open-source mobile robot navigation in crowded environments. Real-time detection, tracking, 3D pose estimation, trajectory prediction. EPFL VITA Lab.
- **Event Camera Fusion** — PyTorch-based model combining event cameras with visual-inertial odometry for 6-DOF drone state estimation. IRI Barcelona.
