abstract: >-
  We present an automated pipeline for city-scale urban tree census using aerial LiDAR
  point clouds and multispectral imagery. The system extracts individual tree morphological
  features — crown diameter, height, canopy volume — and fuses them with spectral
  vegetation indices to predict Leaf Area Index (LAI) via machine learning models,
  enabling scalable canopy density estimation without ground-level surveys.
